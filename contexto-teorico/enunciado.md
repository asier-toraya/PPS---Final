#NO TENER EN CUENTA
Crea una aplicación web usando frameworks (backend fijo, frontend opcional).

Documentación y descripción de resultados en el repositorio individual.

Clínica veterinaria donde tenemos venta de artículos, adopción de mascotas y servicios veterinarios. Roles: admin, clientela, veterinari@s y ventas. Además debe tener ofertas exclusivas en la tienda (artículos y servicios veterinarios) para los clientes que adopten mascotas.

Condiciones:
- Autenticación OAuth 2
- Gestión de autorización RBAC y ABAC 
- Uso de herramientas de SAST (SonarQube)
- Análisis RCA (OWASP dependency check)
- Análisis DAST (OWASP ZAP)
- Desplegar aplicación web usando render.com y supabase
- Usar herramientas de gestión de secretos
